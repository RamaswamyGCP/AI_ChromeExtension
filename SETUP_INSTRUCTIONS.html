2 Download Gemini Nano Model

Wait for the ~1.5GB model to download. This is a one-time setup.

3 Generate Extension Icons

The extension needs 4 icon files. Use the icon generator:

  • Open generate-icons.html in Chrome
  • Click "Download All Icons" button
  • Move downloaded files to /icons folder
  • Verify: icon16.png, icon32.png, icon48.png, icon128.png
4 Load Extension in Chrome

Install the extension:

  • Go to chrome://extensions/
  • Enable "Developer mode" (top-right toggle)
  • Click "Load unpacked"
  • Select this extension folder
  • Pin the extension to your toolbar
5 Test the Extension

Verify everything works:

  • Click the extension icon in toolbar
  • Try the "AI Prompt" tab
  • Enter: "Explain AI in simple terms"
  • Click "Generate" and verify response
  • Select text on any webpage to see toolbar
