ICON GENERATION INSTRUCTIONS
============================

To generate the required PNG icons for this Chrome extension:

Option 1: Use the HTML Icon Generator (Recommended)
---------------------------------------------------
1. Open generate-icons.html in your browser (from the root directory)
2. Click "Download All Icons" button
3. Move the downloaded icons to this /icons directory
4. Ensure files are named: icon16.png, icon32.png, icon48.png, icon128.png

Option 2: Use the SVG file
--------------------------
1. Open icon.svg in a vector graphics editor (Inkscape, Adobe Illustrator, etc.)
2. Export as PNG at these sizes: 16x16, 32x32, 48x48, 128x128
3. Save as: icon16.png, icon32.png, icon48.png, icon128.png

Option 3: Use online tools
---------------------------
1. Upload icon.svg to an online SVG to PNG converter
2. Generate the required sizes
3. Download and rename appropriately

Option 4: Create your own
--------------------------
Design your own icons with these specifications:
- Sizes: 16x16, 32x32, 48x48, 128x128 pixels
- Format: PNG with transparency
- Style: Matches the extension theme (purple gradient recommended)
- Icon suggestion: Sparkle/star with "AI" text

Required files:
- icon16.png
- icon32.png  
- icon48.png
- icon128.png
